# 👗 AI Outfit Stylist

An AI-inspired Outfit Stylist web application that helps users choose the perfect outfit based on different occasions. The app allows users to upload an image or capture one using the camera and receive outfit recommendations with a modern, responsive interface.

## ✨ Features

- 📷 Upload Image
- 📸 Camera Capture
- 👗 Occasion-Based Outfit Recommendation
- 🎨 Color Suggestions
- 👟 Shoes Recommendation
- ⌚ Watch Recommendation
- 👜 Bag Recommendation
- 💇 Hairstyle Recommendation
- ⭐ Outfit Score
- 📈 Fashion Trend Indicator
- 🎤 Voice Assistant
- 🌙 Dark / Light Theme
- 📥 Download Recommendation Card
- ❤️ Save Favourite Outfits
- 📜 Outfit History (Local Storage)
- 📱 Fully Responsive Design
- 💎 Modern Glassmorphism UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- TensorFlow.js
- MediaPipe Tasks Vision
- HTML2Canvas

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/ai-outfit-stylist.git
```

2. Open the project folder.

3. Run **index.html** in your browser.

No installation or backend required.
